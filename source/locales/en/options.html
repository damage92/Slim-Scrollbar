Slim Scrollbar version 0.8-6
by Damage92

To turn off the default scroll bar go in preferences (ctrl+F12) Advanced > browsing, deactivate "show scroll bars".

General options:
Solve slow scrolling - use this option if you are experiencing slow scrolling [set to default value]
Auto hide - auto hide the scrollbars after a few seconds [set to default value]
View bar only when mouse is over it [set to default value]

Bar apparence:
 Bars size:           - the size of the bars (min 3, max 10) [set to default value]
 Bars color:         - color of the bars [set to default value]
 Border color:     - bar's border color [set to default value]
